# Pocket Expense Tracker 💰

A clean, console-based Python application developed as part of the Software Quality Engineering (SQE) curriculum. The project demonstrates a complete software assurance pipeline—spanning manual behavioral verification, automated unit testing, static code quality diagnostics (SonarQube), and vulnerability security scanning (Snyk).

## 🚀 Features
- **Add Expenses:** Record expenditures with positive numeric bounds, categories, and descriptions.
- **View Total Expenses:** Instantly aggregate cumulative costs across all logged data.
- **Filter by Category:** Perform case-insensitive lookups for targeted data evaluation.
- **Admin Security Gate:** Protects configurations using modular validation controls.

## 🛠️ Verification & Quality Infrastructure
- **Unit Testing:** Powered by Python's native `unittest` framework.
- **Static Code Analysis (SCA):** Continuous inspection verified locally via **SonarQube Server & pysonar scanner**.
- **Security Auditing (SAST):** Automated supply-chain and vulnerability validation performed through **Snyk.io**.
